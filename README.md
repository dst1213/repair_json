## Repair JSON（JSON修复工具）

This code is generated by ChatGPT, but it's useful for me.

- incomplete JSON
- missing quotes
- missing brackets
- missing braces


- 截断、不完整的JSON
- 引号缺失
- 中括号缺失
- 花括号缺失

## Examples
```python
    data = """{"name":"tom", "articles":["i'm a boy","""
    res = repair_json(data)
    print(res)
    >>> {"name":"tom", "articles":["i'm a boy"]}
    
    data = """{"name":"tom", "articles":["i'm a bo"""
```
